1e14

About

The 1e14 project aims to bring function graph programming (FGP) into the mainstream.

In "function graph programming", code is organized into a nested structure of interconnected nodes, rather than text files.

We do this by making the building blocks available for creating FGP programs as open-source libraries and frameworks.

Projects

1e14 is running two open-source projects at the moment. We're looking to expand this over time, as FGP lends itself very well to a wide range of applications.

1e14

The first, also named "1e14", is a library that provides the foundation for building FPG programs. It's fairly stable, and well documented.

Ninety

The other one, "ninety", is a modern web application framework.

Ninety is work in progress and is not ready for use.
